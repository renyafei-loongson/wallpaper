应用名称：动态壁纸

开发者：龙芯中科技术有限公司

版本：1.0.1

编程语言：Python

适用操作系统：Loongnix（Fedora21 mips64el）

* 功能简介

每隔一定时间自动切换壁纸。
壁纸来源：在配置文件中指定，可以是本机目录，也可以是一个网上上的FTP目录
